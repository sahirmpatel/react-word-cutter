Simple way to trim text content as per your given limit

## Install

`npm i react-word-cutter --save`

## Import and use

```
import WordCutter from "react-word-cutter"

<WordCutter limit={8} >
Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatem, provident.
</WordCutter>

```
